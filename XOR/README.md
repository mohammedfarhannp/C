# XOR Conversion Program in C language

This program lets user to XOR convert strings either with user input text or specifying file path and a key of any length.

The program saves the output in `Output_XOR.dat`.

# Limitations
 1. Key length maximum is 64 Bytes.
 2. Input Text maximum is 4096 Bytes.
 3. Output cannot be displayed as XOR convertion may turn characters to non printable characters.
 4. Cannot specify a different output file.


# Conclusion
I just wanted to test XOR encryption/decryption. Although python was easier choice I wanted to do it on C as it was a fun experience to do certain stuff from scratch and not using builtIn funcitons.
